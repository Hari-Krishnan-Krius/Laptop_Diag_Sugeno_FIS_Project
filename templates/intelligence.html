What are Linguistic Variables?

A linguistic variable is a variable whose values are words or sentences in a natural language rather than numbers. In our Sugeno FIS, each of the 8 hardware sensor readings is a linguistic variable with membership functions that map crisp numeric sensor values to fuzzy degrees between 0 and 1. The Generalised Bell (gbell) function μ(x) = 1 / (1 + |(x−c)/a|^(2b)) is used for all membership functions.

Output Variable — Fault Class

The output variable is the diagnosed fault class. In the Sugeno FIS, each output class is represented by a crisp singleton (one-hot vector). The final output is the weighted average of all rule outputs, producing a probability-like distribution across the 8 fault classes.

Fuzzy Rule Base

The rule base contains 8 IF-THEN rules, one per fault class. Each rule fires with a strength (weight) between 0 and 1. The final diagnosis is determined by the rule with the highest weighted output after defuzzification. Rule weights are computed using product (AND) of memberships.

Inference Mechanism — Sugeno Method

This system uses the Sugeno (Takagi-Sugeno-Kang) FIS with crisp (singleton) outputs. Unlike Mamdani FIS, the Sugeno method uses crisp output functions which makes defuzzification computationally efficient — ideal for real-time embedded diagnostics.

1
Fuzzification

Each crisp sensor value Xᵢ is passed through its membership function to produce a degree of membership μ ∈ [0, 1].

μ(x) = 1 / (1 + |( x−c ) / a|^2b)

Parameters: a = width, b = slope, c = centre

2
Rule Aggregation

Each rule weight wᵢ is computed by combining the relevant membership values using the product operator (AND):

wᵢ = μ₁(X) × μ₂(X) × ...

The Healthy System rule uses the geometric mean of all 8 nominal memberships to ensure it only dominates when all sensors are simultaneously nominal.

3
Defuzzification

Weighted average of crisp output vectors (one-hot encoding):

y = Σ(wᵢ × zᵢ) / Σwᵢ

The class with the highest score in y is the diagnosis. Confidence = top₁ / (top₁ + top₂).
